### 🚀 Proyecto Destacado

#### *Sistema de Gestión Empresarial para Panadería y Repostería*
Diseño e implementación de un sistema de gestión de inventarios (con lógica de productos perecederos), personal (cálculo de salarios por día), proveedores (artesanales e industriales) y ventas.

*   *Arquitectura Limpia:* Separación estricta de responsabilidades usando el patrón *MVC* para independizar la lógica de negocio de la interfaz de usuario.
*   *Manejo de Datos Profesional:* Implementación de *DTOs* y *DataMappers* para desacoplar las entidades de persistencia de la capa de presentación.
*   *Robustez:* Control de excepciones personalizado (ExceptionChecker) y persistencia de datos mediante archivos de texto y serialización de objetos.
*   👉 Próximamente migrando la lógica del controlador a una API REST con Spring Boot.
